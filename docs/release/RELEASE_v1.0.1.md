# GitHub Release v1.0.1

## Release Title
```
v1.0.1 - 修正打包檔名相容性問題
```

## Release Notes

### 🔧 Bug 修正版本

此版本主要修正打包檔名的相容性問題,確保在 GitHub Releases 中能正常顯示和下載。

---

### 🐛 修正內容

#### 檔名相容性
- **修正前**: `TECO加班計算器.exe` (中文檔名可能在某些環境下出現問題)
- **修正後**: `overtime-assistant-1.0.1.exe` (英文檔名,附帶版本號)

#### 改進優點
- ✅ 更好的跨平台相容性
- ✅ 檔名包含版本號,便於版本管理
- ✅ 符合國際化命名規範
- ✅ 避免 GitHub Releases 可能的編碼問題

---

### 📦 安裝與使用

#### 下載
請從本頁面的 Assets 區下載 `overtime-assistant-1.0.1.exe`

#### 使用方式
1. 下載 `overtime-assistant-1.0.1.exe`
2. 雙擊執行 (與 v1.0.0 使用方式完全相同)
3. 功能完全一致,僅檔名不同

---

### 🔄 從 v1.0.0 升級

如果您已經在使用 v1.0.0:

1. **無需更新**: v1.0.1 僅修正檔名,功能完全相同
2. **選擇性更新**: 如果您希望統一檔名規範,可以下載新版本
3. **直接替換**: 下載後直接替換舊版執行檔即可

**重要提示**: 您的設定、匯出的報表等資料不受影響。
