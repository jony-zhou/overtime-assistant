# GitHub Release v1.1.1

## Release Title
```
v1.1.1 - 圖示解析度優化
```

## Release Notes

### 🎨 視覺優化：提升圖示解析度

此版本針對應用程式圖示進行優化,提升顯示品質,特別是在高解析度螢幕上的表現。

---

## ✨ 改進內容

### 圖示解析度提升 🖼️

#### 主要變更
- ✅ **原始圖示尺寸**: 15x15 → **128x128** (提升 8.5 倍)
- ✅ **登入頁面圖示**: 使用 128x128 原始圖片,CTkImage 縮放至 64x64
- ✅ **導覽列 Logo**: 使用 128x128 原始圖片,CTkImage 縮放至 32x32

#### 優化效果
- ✅ **高 DPI 支援**: 在 4K/Retina 顯示器上更清晰
- ✅ **平滑縮放**: CTkImage 自動處理高品質縮放
- ✅ **視覺一致性**: 登入和主視窗圖示品質統一

#### 技術細節
```python
# 登入頁面 (64x64 顯示)
ctk_image = ctk.CTkImage(
    light_image=img,  # 128x128 原始圖
    dark_image=img,
    size=(64, 64)     # 顯示尺寸
)

# 導覽列 (32x32 顯示)
ctk_image = ctk.CTkImage(
    light_image=img,  # 128x128 原始圖
    dark_image=img,
    size=(32, 32)     # 顯示尺寸
)
```

---

## 🔧 技術說明

### 為什麼使用 128x128?

1. **高 DPI 準備**: 支援 2x/3x 縮放顯示
2. **品質保證**: 原始圖片大於顯示尺寸,確保縮放品質
3. **標準尺寸**: 128x128 是常見的圖示標準尺寸
4. **向下相容**: 可縮放至任意尺寸而不失真

### 影響範圍

- ✅ 登入頁面應用程式圖示
- ✅ 主視窗導覽列 Logo
- ✅ 視窗標題列圖示 (icon.ico)
